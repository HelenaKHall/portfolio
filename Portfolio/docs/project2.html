1.5 Question 5: Executive Summary Creation

1.5.1 Create a comprehensive business summary by calculating: total revenue across all regions and quarters, average revenue per quarter, and the number of high-performance records (revenue > $140,000). Present these as a formatted table or clear summary.

# Total revenue
total_revenue <- sum(regional_sales$revenue)

# Average revenue per quarter
avg_revenue_quarter <- aggregate(revenue ~ quarter, data = regional_sales, mean)

# Number of high-performance records (revenue > 140000)
n_high_perf <- sum(regional_sales$revenue > 140000)

# Create a summary table
business_summary <- data.frame(
  Metric = c("Total Revenue", 
             paste("Average Revenue", avg_revenue_quarter$quarter, sep=" - "), 
             "High-Performance Records (Revenue > $140,000)"),
  Value = c(total_revenue, round(avg_revenue_quarter$revenue, 2), n_high_perf)
)
